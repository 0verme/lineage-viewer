## Status definitions

- **Completed:** accepted work with its required verification.
- **Current:** the active phase.
- **Planned:** intended work that has not started.
- **Deferred:** deliberately postponed pending a stable core API and demonstrated need.
- **Out of scope:** capabilities outside the viewer boundary, including extraction, storage, governance, authentication, and general graph editing.
